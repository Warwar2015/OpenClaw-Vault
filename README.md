# 🧠 OpenClaw-Vault

My personal knowledge base with OpenClaw AI integration.

## 📁 Structure
- **Daily/** - Daily notes and journal entries
- **Notes/** - Permanent notes (Zettelkasten style)
- **Knowledge/** - Topical knowledge organization
- **Inbox/** - Quick captures and temporary notes
- **Projects/** - Project-specific documentation
- **Templates/** - Note templates

## 🤖 OpenClaw Integration
This vault is synchronized with OpenClaw AI assistant for:
- Automatic note creation
- Smart search across all notes
- AI-powered knowledge extraction
- Automated workflows

## 🔄 Sync
Automatically synced via GitHub. OpenClaw runs on server and manages notes.
